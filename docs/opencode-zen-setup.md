# OpenCode Zen / Go Setup

This project uses OpenCode-hosted models, not Ollama.

1. Run `opencode`
2. Run `/connect`
3. Choose `OpenCode Zen` or `OpenCode Go`
4. Add your OpenCode API key
5. Run `/models`
6. If your account does not show `opencode/gpt-5.1-codex` or `opencode/gpt-5`, replace model IDs in `opencode.json`.

Start:

```text
/start-collabify
```
