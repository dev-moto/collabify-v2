# React Rules

- Use React JS with Vite.
- Use functional components.
- Use Redux Toolkit for global auth/session/profile state.
- Use service modules for Supabase queries.
- Use TailwindCSS for styling.
- Use lucide-react for icons.
- Use React Router for routing.
- Every async page needs loading, error, and empty states.
- Every form needs validation.
- Use accessible HTML.
- Never read or display raw private fields unless allowed by role and RLS.
