# Threat Model Prompt

Create a threat model for Collabify.

Cover:
- Account takeover
- Fake creators
- Fake businesses
- Document fraud
- Scraping creator data
- Spam messages
- Appointment abuse
- File upload malware
- RLS bypass
- Service role leaks
- GitHub secret leaks
- VPN bypass
- Data privacy incidents

Use severity, likelihood, mitigation, owner, and test coverage.
